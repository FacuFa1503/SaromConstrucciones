# Sarom Constructions 🏗️
**Building Excellence**

Proyecto de desarrollo web para Sarom Constructions, enfocado en una interfaz moderna, limpia y responsive para mostrar obras y servicios de construcción.

## 🚀 Tecnologías utilizadas
* **HTML5** & **SASS** (Arquitectura de estilos avanzada).
* **Bootstrap 5** (Layout y grillas).
* **Google Fonts** (Tipografía Gotham).
* **Instagram API Embed** (Visualización de obras en tiempo real).

## 🛠️ Características
* **Hero Banner Cover**: Diseño adaptativo que mantiene la integridad del logo en todas las resoluciones.
* **Glassmorphism Nav**: Barra de navegación con efecto de desenfoque dinámico.
* **Responsive Grid**: Galería de Instagram configurada en 2x2 para móviles y 3x3 para desktop.
* **Fondo Estático**: Gradiente radial fijo para una experiencia de usuario fluida.

## 📐 Instalación y Desarrollo
1. Clona el repositorio.
2. Si realizas cambios en los estilos, asegúrate de compilar el archivo `scss/main.scss` hacia `css/main.css`.
